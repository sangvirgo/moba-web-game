# ⚔️ MOBA Arena

A complete Multiplayer Online Battle Arena (MOBA) web game with AI bots, built entirely in a single HTML file.

## 🎮 Play Now

**[Play MOBA Arena](https://sangvirgo.github.io/moba-web-game/)**

## Features

- **5 Playable Heroes** with unique abilities (Tank, Mage, Assassin, Marksman, Support)
- **3 Lanes** with towers, minion waves, and jungle monsters
- **AI Bots** that lane, fight, retreat, and buy items
- **Fog of War** with limited vision range
- **Item Shop** with 16 items
- **Level-up System** (max level 15)
- **Minimap** showing the full battlefield
- **Responsive UI** with mobile touch controls

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| Q / W / E / R | Use abilities |
| Shift + Q/W/E/R | Level up ability |
| B / P | Open shop |
| Space | Stop movement |
